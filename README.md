# Outil LPA — 1er RTP
**BML du 1er RTP – LTN LAOUAR**

Application web de consultation des conditionnements de largage aéroporté.
Basée sur le Manuel LPA 2020 et le Tableau PIA.

## Utilisation locale
Ouvrir `index.html` directement dans Firefox. Aucune connexion internet requise.

## Mise en ligne GitHub Pages
1. Créer un dépôt GitHub (public ou privé)
2. Uploader le contenu de ce dossier
3. Settings → Pages → branche main → Save
4. Accès : `https://[compte].github.io/[depot]/`

## Données intégrées
- Manuel LPA 2020 complet (gravité FH/MH/GH/TGH/droppage/wedge/autoguidé)
- Éjection FH/MH/GH mer (LTCO11/12/13)
- Omnibus — Colis de ravitaillement RVA
- Capacités A400M (LOMX/LOLEX), C130H/H30, C130J, CASA235-200/300
- Hélicoptères : Puma, Caracal, Caïman, PC-6
- Transall C-160 exclu (hors service)
